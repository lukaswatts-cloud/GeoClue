# GeoClue — Frontend

This folder holds everything players see and interact with: maps, clues, scores, and location permissions.

## What goes here later

- **UI** — screens, buttons, and layouts for the game
- **Map / location** — showing the player on a map and nearby clues
- **API calls** — talking to the backend to load game data and submit answers

## Folders

| Path | Purpose |
|------|---------|
| `src/` | Application source code (you’ll add components, pages, and logic here) |

When you pick a framework (e.g. React, Vue, or plain HTML/JS), most of your code will live under `src/`.
