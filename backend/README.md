# GeoClue — Backend

This folder holds the server: game rules, storing clues and player progress, and validating location-based actions.

## What goes here later

- **HTTP API** — routes the frontend calls (e.g. “get nearby clues”, “submit guess”)
- **Data** — database or files for clues, users, and sessions (you’ll choose how to store data)
- **Auth** (optional) — sign-in if you want accounts

## Folders

| Path | Purpose |
|------|---------|
| `src/` | Server source code (routes, game logic, database access) |

You can use any language you like (Node, Python, Go, etc.); this repo only defines a simple layout for now.
